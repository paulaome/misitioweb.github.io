Llenar un formulario no es una tarea agradable, se puede hacer que el formulario se vea mucho mas atractivo de usar y de esta forma mas personas lo diligenciaran. Tambien, algunos navegadores modifican su apariencia y los visualizan de forma diferente. CSS se utiliza comunmente para controlar la apariencia de los elementos de formulario. Esto es para hacerlos mas atractivos y hacerlos mas consistentes en diferentes navegadores.
