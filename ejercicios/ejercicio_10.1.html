EH HTML5 no es necesario proporcionar valores para todos los atributos, como los controles, reproduccion automatica y atributos de bucle utilizados con el elemento <video>. Estos atributos son como interruptores de encendido apagado. Si el atributo esta presente, activa esa opcion. Si se omite el atributo, la opcion esta desactivada. Si el navegador no admite el elemento <Video> o el formato de video utilizado, se mostrara lo que este entre la <video> de apertura y las etiquetas de </video> de cierre
