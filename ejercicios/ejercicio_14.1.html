Forma predeterminada, un cuadro tiene el tamaño lo sufucientemente grande para mantener su contenido. Para establecer las dimensiones de un cuadrado, se puede utilizar las propiedades: height y width. Las formas mas populasres de especificar el tamaño de un cuadro son usar pixeles, prcentajes o EMS. Tradicionalmente, los pixeles hay sido el metodo mas popular, ya que permiten controlar con precision su tamaño. En este ejemplo se puede ver que se utiliza un elemento < div > que contiene 400 pixeles de ancho por 400 pixeles de alto.
