Background color- color de fondo

Si no se especifica un color de fondo, el fondo sera transparente.

CSS trata cada elemento HTML como un cuadro y la prioridad background-color establece el Color del fondo para el cuadro. Se puede especificar un color de fondo de las mismas tres formas en que se puede especificar colores de primer plano: Valores RGB, codigo hexadecimal y nombre de colores. Tambien hemos utilizado las propiedades padding para separar el texto de los bordes de los cuadros. Esto hace que sea mas facil de leer.
