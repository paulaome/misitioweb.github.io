Letter and word spacing

by TPS Sena

El kerning es el termino que utilizan los tipografos para el espacio entre cada letra. Puede controlar el espacio entre cada letra con la propiedad: letter-spacing. Es particularmente util para aumentar el kerning cuando su encabezado o frase esta en mayuscula. Si el texto esta en un caso o normal, aumentar o disminuir el kering puede dificultad la lectura.

El intervalo preterminado

Cuando se especifica un valor para estas propiedades, se debe dar en EMS. El intervalo predeterminado entre las palabras se establece por el tipo de letra (a menudo alrededor de 0,25 EM), y es poco probable que deba cambiar esta propiedad con regularidad.
